# Breast-Cancer-prediction-AI-model
## Overview
This project is an AI-powered web application that predicts the likelihood of breast cancer using common diagnostic metrics. It provides real-time probability estimates for malignant tumors, helping users understand their health data in a simple, interactive format.

## Features
Easy Input: Enter medical data such as radius, texture, perimeter, area, and more.

Instant Prediction: Get quick probability scores for malignant diagnosis.

Radar Chart Visualization: See your input and feature strengths in an interactive chart.

Educational Warning: Clearly states that the prediction is not a substitute for professional medical diagnosis.

## Tech Stack
- Python

- scikit-learn (machine learning)

- pandas (data processing)

- Streamlit (web app & UI)

- Plotly (data visualization)

- Custom CSS (styling)
